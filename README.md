## Netflix Content Analysis Dashboard
This project presents an interactive Power BI dashboard that analyzes Netflix Movies and TV Shows to uncover trends in content distribution, ratings, and global availability.
---
### Project Overview
The dashboard provides insights into Netflix content using interactive visualizations and filters. Users can explore trends across genres, countries, ratings, and release years.
---
### Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning
- Interactive Dashboard Design
---
### Key Insights
- Netflix content production increased significantly after 2016.
- International Movies represent the largest category on the platform.
- The United States and India contribute a large portion of Netflix content.
---
### Dashboard Features
- Interactive slicers for Movies and TV Shows
- Global map visualization showing content distribution by country
- Content growth analysis over time
- Top categories and ratings analysis
---
### Project Files
- `Netflix_Dashboard.pbix` – Power BI dashboard file
- `netflix_Dataset.csv` – Dataset used for analysis
